# IntelliJ IDEA

## Installation

1. Install IntelliJ IDEA Ultimate Edition

    ```sh
    brew cask install intellij-idea
    ```
1. Just create a symlink for IdeaVim's configuration.

    ``` sh
    ln -fs ~/.dotfiles/intellij-idea/ideavimrc ~/.ideavimrc
    ```
